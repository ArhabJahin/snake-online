# 🐍 Snake Online

**Snake Online** is a real-time multiplayer version of the classic Snake game built for the web. Players can join the same game session, compete against each other, and test their reflexes in a fast-paced environment where every move matters.

The game recreates the nostalgic Snake experience while introducing online multiplayer gameplay, allowing multiple players to control their own snakes simultaneously. As players collect food, their snakes grow longer, increasing both their score and the challenge of avoiding collisions.

## Features

* 🎮 Real-time multiplayer gameplay
* 🌐 Browser-based game with no installation required
* 🐍 Classic Snake mechanics with smooth controls
* 🍎 Food collection and snake growth
* 🏆 Live score tracking
* ⚡ Fast and responsive gameplay
* 🔄 Automatic game state synchronization between players

## Gameplay

* Use the keyboard controls to move your snake.
* Eat food to increase your score and grow longer.
* Avoid colliding with walls, your own body, and other players' snakes.
* Stay alive longer and earn the highest score to dominate the leaderboard.

## Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js
* **Real-time Communication:** WebSockets / Socket.IO
* **Version Control:** Git & GitHub

## Project Objective

This project was developed to explore real-time multiplayer game development using modern web technologies. It demonstrates concepts such as client-server communication, game state synchronization, collision detection, and responsive browser-based gameplay.

Whether you're playing for fun or exploring the source code, **Snake Online** showcases how a classic arcade game can be transformed into a competitive online multiplayer experience.
